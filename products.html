Release Notes

v2.1 Revit 2025.3+
  • Room Acoustics refinements: Improved surface discovery for stacked floor finishes, operable partitions in linked models, and divisible-room boundaries; cleaner Assign Materials grid; reference curve overlay on the RT60 graph for comparison with EASE / ODEON / measured field data.
  • Schedule preservation: Running RT60 on rooms at one level no longer wipes calculations for rooms at other levels.
  • Configuration tab: New per-room Nudge button for fine-tuning speaker placement using Revit's native arrow-key nudge.
  • Setup & families: Bundled JSBA tag and coordination families auto-load on first project use; new Circuit ID + Tap tag variant for installation drawings.
  • Lucius AI: Comprehensive knowledge-base update covering every tab, calculation formula, and family setup detail.

v1.2.1 Revit 2022–2024 · Free
  • Linked model stability: Improved tile-centering reliability in linked-room workflows.
  • Project-unit-aware reporting: Column headers and numeric reporting reflect the active Revit project length units (Imperial or Metric).
  • JSBA Control Data schedule: Multi-category schedule creation and population improvements.
